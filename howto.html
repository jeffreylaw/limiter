How to Use

With Limiter, you can set the maximum number of visits to a website allowed in a set period of time. If you exceed the limit, you will blocked from visiting the domain until the timer is up.

Steps

  1. Visit the domain you want to limit and click on the extension icon.
  2. Enter the maximum number of visits you want to allow.
  3. Enter how many hours and minutes you want the Limiter to last.
  4. Click Limit to start the timer.

Other Tips

  • By setting the maximum number of visits to 0, your next visit to the domain will be blocked.
  • Limiter works by blocking the domain, if you create a Limiter on www.facebook.com, any visit to a page under that domain such as www.facebook.com/profile will count as a page visit.
  • If you really need to access a blocked website, you can release the domain by clicking the extension icon while visiting the page.
  • Subdomains are treated individually; i.e. docs.github.com is different than github.com.
